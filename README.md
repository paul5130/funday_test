# Funday Test App

## 專案介紹
- 播放與下載音檔
- 分頁載入（上滑加載更多）
- 使用 Riverpod + MVVM 架構管理狀態

## 第三方套件
- **Flutter** 3.35.4 / **Dart** 3.9.2
- **Riverpod** (狀態管理)
- **Dio** (API 請求)
- **Freezed** (資料模型 / 狀態)
- **just_audio**  (音檔播放)

## 功能說明
- **音檔列表**：從台北旅遊 API 讀取音檔
- **上滑加載更多**：滾動到底部會自動載入下一頁
- **下載功能**：下載 mp3 至本機資料夾
- **播放功能**：已下載的顯示播放按鈕，未下載的顯示下載按鈕
